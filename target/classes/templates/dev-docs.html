AI / ML MODULES

Intelligent Core

🎯

Fuzzy Logic Scorer

Rule-Based
What it does: Scores investor risk from 0–100 using 4 weighted inputs
Inputs: Horizon (25%) · Income (20%) · Experience (20%) · Risk (35%)
Why fuzzy logic: Handles overlapping boundaries better than hard cutoffs
🔵

K-Means Clustering

Unsupervised
What it does: Assigns any stock to a volatility cluster at runtime
How: Computes StdDev, ATR, Max Drawdown → finds nearest centroid
Centroids: Pre-trained on 24 NSE stocks, saved as centroids.json
🌳

Decision Tree Rules

Supervised
What it does: Predicts if a stock suits the investor's risk profile
Inputs: Investor category + Stock volatility cluster
Rules: ML-derived, compiled into Java if-else; ~88% test accuracy
🤖

Gemini NLP Insight

Generative AI
What it does: Converts structured stats into a plain-English explanation
Inputs injected: Symbol, volatility, trend, 1Y return, compatibility
Constraints & Safety: Max 80 words, educational only, appends disclaimer
